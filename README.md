# CASA Data Science for Spatial Systems Assessment 2025/2025
CASA 0006 Data Science for Spatial Systems 2025/2026

This repository is a collection of files for the final assessment for the CASA 0006 Data Science for Spatial Systems module by Dr Huanfa Chen

This individual assignment aims to test the student's ability to conduct in-depth spatial data analysis using methods from this module. The studyent is required to submit a single Python Notebook which contains both the code required to conduct the data analysis and accompanying text which provides context and interpretation.

The requirements are set out [here](https://huanfachen.github.io/DSSS_2025/assessments/)  
The marking scheme can be found [here](https://huanfachen.github.io/DSSS_2025/assessments/#marking-scheme)  
The individual assessment can be found [here](https://benjamintee.github.io/CASA_UDV_Assessment/Energy_Consumption.html)
